## 📌 Recap: What Is an Adjacency Matrix?

Given an `n x n` matrix like this:

```java
int[][] isConnected = {
    {1, 1, 0},
    {1, 1, 0},
    {0, 0, 1}
};
```

* Each row/column represents a node.
* `isConnected[i][j] == 1` means there's an edge between node `i` and node `j`.
* The matrix is **symmetric** for an **undirected graph**.
* Diagonal entries are often 1, since each node is connected to itself.

---

## 🧭 n4_Traversal Techniques.DFS.DFS (Depth-First Search)

### ✅ Recursive n4_Traversal Techniques.DFS.DFS

```java
void dfs(int[][] isConnected, boolean[] visited, int node) {
    visited[node] = true;

    for (int j = 0; j < isConnected.length; j++) {
        if (isConnected[node][j] == 1 && !visited[j]) {
            dfs(isConnected, visited, j);
        }
    }
}
```

### 📌 Use Case Example (Counting Components)

```java
int findCircleNum(int[][] isConnected) {
    int n = isConnected.length;
    boolean[] visited = new boolean[n];
    int count = 0;

    for (int i = 0; i < n; i++) {
        if (!visited[i]) {
            dfs(isConnected, visited, i);
            count++;
        }
    }

    return count;
}
```

## 🚍 n4_Traversal Techniques.BFS.BFS (Breadth-First Search)

### ✅ Iterative n4_Traversal Techniques.BFS.BFS Using Queue

```java
void bfs(int[][] isConnected, boolean[] visited, int start) {
    Queue<Integer> queue = new LinkedList<>();
    queue.offer(start);
    visited[start] = true;

    while (!queue.isEmpty()) {
        int node = queue.poll();

        for (int j = 0; j < isConnected.length; j++) {
            if (isConnected[node][j] == 1 && !visited[j]) {
                queue.offer(j);
                visited[j] = true;
            }
        }
    }
}
```

### 📌 Use Case Example (Counting Components)

```java
int findCircleNum(int[][] isConnected) {
    int n = isConnected.length;
    boolean[] visited = new boolean[n];
    int count = 0;

    for (int i = 0; i < n; i++) {
        if (!visited[i]) {
            bfs(isConnected, visited, i);
            count++;
        }
    }

    return count;
}
```
