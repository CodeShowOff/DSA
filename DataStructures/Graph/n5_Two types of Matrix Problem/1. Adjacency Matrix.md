## ✅ 1. What Is an Adjacency Matrix?

An **adjacency matrix** is a 2D array used to represent a graph. It tells you which nodes are directly connected.

### 📐 Structure

For a graph with **n nodes**, the adjacency matrix is:

```java
int[][] matrix = new int[n][n];
```

* `matrix[i][j] == 1` means there is an **edge from node i to node j**
* `matrix[i][j] == 0` means there is **no edge**

### 🔢 Rows and Columns Meaning

* Total number of **rows = number of nodes = n**
* Each **row `i`** represents **node i**
* Each **column `j` in row i** tells whether **node i is connected to node j**

So:

> **matrix\[i]\[j] == 1** → node `i` is connected to node `j`

---

## 🟩 Example

Let's say we have this graph:

```
    0
   / \
  1   2

Edges: [0-1], [0-2]
```

### Its adjacency matrix (for `n = 3`):

```
   0 1 2
0 [1 1 1]
1 [1 1 0]
2 [1 0 1]
```

### Explanation:

* Row 0: `1 1 1` → node 0 is connected to 0 (itself), 1, and 2
* Row 1: `1 1 0` → node 1 is connected to 0 and 1 (itself)
* Row 2: `1 0 1` → node 2 is connected to 0 and 2 (itself)

---

## 🧠 Important Properties

### 1. **Size** is always `n x n`

Because you check connectivity between **each pair** of nodes.

### 2. **Diagonal elements are 1**

Usually, if the graph is **self-connected** (each node is connected to itself), you’ll see:

```
matrix[i][i] = 1
```

But it’s not **strictly required** — in some problems, it could be 0 if self-loops are not allowed. In the **“Number of Provinces”** problem, each node is connected to itself, so diagonal is `1`.

### 3. **Undirected Graph: matrix\[i]\[j] == matrix\[j]\[i]**

This is a **symmetric matrix**, because connection is mutual:

If `0 ↔ 1`, then:

```
matrix[0][1] = 1
matrix[1][0] = 1
```

In a **directed graph**, this property **doesn’t hold**.

---

## 🗂 Why We Create a Visited Array of Size `n`?

Even though the matrix has `n x n` entries, you only have **n nodes** (from 0 to `n-1`). The matrix describes **relationships**, but the actual **entities to visit** are just the nodes.

So:

```java
boolean[] visited = new boolean[n]; // true means the node has already been visited
```

---

## 🔁 Traversal Idea Using Matrix

Let’s say you want to find all neighbors of node `i`:

```java
for (int j = 0; j < n; j++) {
    if (matrix[i][j] == 1 && !visited[j]) {
        // node j is a neighbor of i and not visited yet
    }
}
```

## 🧪 Full Example

### Graph:

```
    0
   / \
  1   2

Only one connected component (all connected)
```

### Matrix:

```java
int[][] isConnected = {
    {1, 1, 1},
    {1, 1, 0},
    {1, 0, 1}
};
```

### n4_Traversal Techniques.DFS.DFS:

```java
void dfs(int[][] isConnected, boolean[] visited, int i) {
    visited[i] = true;
    for (int j = 0; j < isConnected.length; j++) {
        if (isConnected[i][j] == 1 && !visited[j]) {
            dfs(isConnected, visited, j);
        }
    }
}
```

### Count provinces:

```java
int count = 0;
for (int i = 0; i < n; i++) {
    if (!visited[i]) {
        count++;
        dfs(isConnected, visited, i);
    }
}
```

## 🚨 Common Mistakes to Avoid

| Mistake                                                         | Why it’s wrong                                                |
| --------------------------------------------------------------- | ------------------------------------------------------------- |
| Thinking `matrix[i][j]` is an edge like `(i, j)`                | It’s just a flag: connected or not                            |
| Using nested for-loops on `matrix[i][j]` assuming they’re edges | You should loop through nodes and check neighbors             |
| Not marking visited properly                                    | Leads to infinite recursion or overcounting                   |
| Confusing adjacency matrix with grid problems                   | In grid, adjacency is by **position**, not logical node index |

## 📌 Summary

| Concept                      | Meaning                                    |
| ---------------------------- | ------------------------------------------ |
| Rows / Columns               | Each corresponds to a node                 |
| `matrix[i][j] == 1`          | Node `i` is connected to node `j`          |
| Diagonal `matrix[i][i] == 1` | Node connected to itself                   |
| Symmetric (`i ↔ j`)          | Undirected graph                           |
| `n x n` matrix               | Because each node can connect to any other |
| `visited[n]`                 | You only have `n` actual nodes, not `n²`   |
